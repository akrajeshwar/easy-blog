**EasyBlog** is a cloud-based blogging platform built using a **serverless architecture**.  
The frontend is hosted on **GitHub Pages**, while **Firebase Firestore** is used as a cloud database to store and retrieve blog posts dynamically.

The project allows users to:
- Write and submit blog posts
- Store blogs in the cloud
- View all blogs dynamically
- Read individual blog posts

No traditional backend server is used, making the project lightweight, scalable, and cost-free.
